# Flask Backend

This is a simple Flask backend for your project.

## Usage

1. Create and activate a virtual environment:
   ```sh
   python -m venv venv
   venv\Scripts\activate  # On Windows
   ```
2. Install Flask:
   ```sh
   pip install flask
   ```
3. Run the server:
   ```sh
   python app.py
   ```

## API Endpoints
- `GET /api/hello` - Returns a hello message.
